1) Criar um sistema para cadastrar, selecionar e excluir produtos. Crie uma fake api com as seguintes características: nome do produto, valor, marca e id.

2) Implementar um sistema para cadastrar receitas. Deverá ter as seguintes funcionalidades:
> Cadastrar receitas
> Listar todas as receitas (exibir o nome e um botão saiba mais)
> Receita completa (quando clicar no botão saiba mais, exibir todos os dados da receita).

As informações da API serão: nome da receita, ingredientes, modo de preparo, quantidade de pessoas que a receita serve e o id.

3) Criar uma API com as características: artista, musica, letra e id.

Sempre que iniciado o front, deverá ser feito um método GET para armazenar TODOS os dados da fake api em um JSON local.

O usuário terá dois campos para preencher, um campo de artista e outro com o nome da música. Quando pesquisar retorne a letra da música, caso contrário retorne que não foi encontrada a canção.
